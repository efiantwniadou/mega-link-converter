## 🧾 What this app does

mega-link-converter helps you turn Mega.nz links into direct files, videos, and streamable media through a Telegram bot.

Use it when you want to:

- download files from Mega links without an account
- convert Mega links into shareable downloads
- send large files through Telegram
- split big files into smaller parts
- add custom captions and thumbnails
- keep downloads moving with parallel tasks

## 💻 What you need

Use a Windows PC with:

- Windows 10 or Windows 11
- a stable internet connection
- enough free disk space for the file you want to download
- Telegram installed on your phone or desktop
- a modern browser to open the release page

For large files, keep extra free space on your drive. The app may create temporary files while it works.

## 🛠️ First-time setup

After you open the app, set up the basics:

1. Start the program
2. Paste your Mega.nz link into the input box
3. Choose the output type you want, such as file, video, or streamable media
4. Set your Telegram bot details if the app asks for them
5. Pick the folder where you want downloads saved
6. Add a caption if you want one
7. Add a thumbnail if you want one
8. Start the job

If the app uses a bot token, copy it from Telegram and paste it into the setup screen. If it asks for a chat ID, use the one tied to your bot chat.

To avoid problems:
